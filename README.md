## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Clone Repository

```bash
$ git clone https://github.com/Mark0-22/Unit-Converter.git
cd Unit-Converter
```

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
